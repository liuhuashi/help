openssl_x509_free

(PHP 4 >= 4.0.6, PHP 5, PHP 7, PHP 8)

openssl_x509_free — 释放证书资源

说明

openssl_x509_free ( resource $x509cert ) : void

openssl_x509_free() 从内存中释放和指定 x509cert资源相关联的证书。

参数

x509cert

返回值

没有返回值。
