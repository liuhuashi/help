OAuth::setCAPath

(PECL OAuth >= 0.99.8)

OAuth::setCAPath — 设置 CA 路径和信息

说明

public OAuth::setCAPath ( string $ca_path = ? , string $ca_info = ? ) : mixed

设置证书授权中心（CA ）的路径和信息。

警告

本函数还未编写文档，仅有参数列表。

参数

ca_path

要设置的 CA 路径。

ca_info

要设置的 CA 信息。

返回值

成功则返回 true ，如果 ca_path 或 ca_info 其中之一被认为无效则返回 false 。

更新日志

版本 说明
1.0.0 以前失败时返回 null，而不是 false。
