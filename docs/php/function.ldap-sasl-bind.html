ldap_sasl_bind

(PHP 5, PHP 7, PHP 8)

ldap_sasl_bind — Bind to LDAP directory using SASL

说明

ldap_sasl_bind ( resource $ldap , string|null $dn = null , string|null $password = null , string|null $mech = null , string|null $realm = null , string|null $authc_id = null , string|null $authz_id = null , string|null $props = null ) : bool
警告

本函数还未编写文档，仅有参数列表。

返回值

成功时返回 true， 或者在失败时返回 false。

更新日志

版本 说明
8.0.0 dn, password, mech, realm, authc_id, authz_id and props are nullable now.

注释

注意: Requirement
ldap_sasl_bind() requires SASL support (sasl.h). Be sure --with-ldap-sasl is used when configuring PHP otherwise this function will be undefined.
