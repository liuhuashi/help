SolrDocument::__isset

(PECL solr >= 0.9.2)

SolrDocument::__isset — Checks if a field exists

说明

public SolrDocument::__isset ( string $fieldName ) : bool

Checks if a field exists

参数

fieldName

Name of the field.

返回值

成功时返回 true， 或者在失败时返回 false。
