zip_entry_name

(PHP 4 >= 4.1.0, PHP 5 >= 5.2.0, PHP 7, PHP 8, PECL zip >= 1.0.0)

zip_entry_name — 检索目录项的名称

说明

zip_entry_name ( resource $zip_entry ) : string

返回指定目录项的名称。

参数

zip_entry

由函数zip_read() 返回的目录项。

返回值

目录项的名称。

参见

  • zip_open() - 打开ZIP存档文件
  • zip_read() - 读取ZIP存档文件中下一项
