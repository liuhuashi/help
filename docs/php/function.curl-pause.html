curl_pause

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

curl_pause — 暂停和取消暂停一个连接。

说明

curl_pause ( resource $ch , int $bitmask ) : int

警告

本函数还未编写文档，仅有参数列表。

参数

handle

由 curl_init() 返回的 cURL 句柄。

bitmask

CURLPAUSE_* 常量之一。

返回值

返回一个错误代码 (如果没有错误则返回CURLE_OK常量)。
