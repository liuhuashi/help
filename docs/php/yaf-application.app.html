Yaf_Application::app

(Yaf >=1.0.0)

Yaf_Application::app — 获取当前的Yaf_Application实例

说明

public static Yaf_Application::app ( ) : void

返回Yaf_Application的实例， 也可以使用Yaf_Dispatcher::getApplication()来得到Yaf_Application的实例 Yaf_Dispatcher::getApplication().

参数

此函数没有参数。

返回值

返回值为一个Yaf_Application的实例， 如果在调用之前没有初始化一个Yaf_Application实例的话，它将返回NULL

参见

  • Yaf_Dispatcher::getApplication() - 获取当前的Yaf_Application实例
