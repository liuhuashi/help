hash_algos

(PHP 5 >= 5.1.2, PHP 7, PHP 8, PECL hash >= 1.1)

hash_algos — 返回已注册的哈希算法列表

说明

hash_algos ( ) : array

返回值

返回一个数值索引的数组， 包含了受支持的哈希算法名称。

更新日志

版本 说明
7.4.0 支持 crc32c。
7.1.0 加入 sha512/224，sha512/256，sha3-224，sha3-256，sha3-384 以及 sha3-512 算法的支持。

范例

示例 #1 hash_algos() 例程

print_r(hash_algos());
