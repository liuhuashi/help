mcrypt_enc_get_block_size

(PHP 4 >= 4.0.2, PHP 5, PHP 7 < 7.2.0, PECL mcrypt >= 1.0.0)

mcrypt_enc_get_block_size — 返回打开的算法的分组大小

警告

本函数已自 PHP 7.1.0 起废弃并将自 PHP 7.2.0 起移除。强烈建议不要使用本函数。

说明

mcrypt_enc_get_block_size ( resource $td ) : int

获取打开的算法的分组大小。

参数

td

加密描述符。

返回值

返回指定算法的分组大小，以字节为单位。
