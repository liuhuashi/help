Error::getLine

(PHP 7, PHP 8)

Error::getLine — 获取错误发生时的行号

说明

final public Error::getLine ( ) : int

获取错误发生时的行号。

参数

此函数没有参数。

返回值

返回错误发生时的行号。

范例

示例 #1 Error::getLine() 例子

<?php
try {
    throw new Error("Some error message");
} catch(Error $e) {
    echo "The error was created on line: " . $e->getLine();
}
?>

以上例程的输出类似于：

The error was created on line: 3
