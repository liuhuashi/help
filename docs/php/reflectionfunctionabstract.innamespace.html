ReflectionFunctionAbstract::inNamespace

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

ReflectionFunctionAbstract::inNamespace — 检查是否处于命名空间

说明

public ReflectionFunctionAbstract::inNamespace ( ) : bool

检查函数是否在命名空间内定义

警告

本函数还未编写文档，仅有参数列表。

参数

此函数没有参数。

返回值

在命名空间内返回 true，否则返回 false
