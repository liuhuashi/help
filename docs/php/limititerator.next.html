(PHP 5 >= 5.1.0, PHP 7, PHP 8)

LimitIterator::next — Move the iterator forward

说明

public LimitIterator::next ( ) : void

Moves the iterator forward.

参数

此函数没有参数。

返回值

没有返回值。
