SplFileObject::getChildren

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

SplFileObject::getChildren — No purpose

说明

public SplFileObject::getChildren ( ) : void

An SplFileObject does not have children so this method returns null.

参数

此函数没有参数。

返回值

没有返回值。
